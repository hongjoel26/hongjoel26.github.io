Autonomous Mobile Robot

An autonomous mobile robot developed to transport workpieces between manufacturing stations using a mecanum drive platform and intelligent navigation.

Project Overview

The Autonomous Mobile Robot (AMR) was developed as part of a Smart Factory simulation to transport workpieces between manufacturing stations autonomously using a mecanum-wheel drive platform.

Key Responsibilities

  • Developed autonomous navigation logic.
  • Integrated mecanum wheel motion control.
  • Implemented sensor-based navigation.
  • Tested transportation accuracy.

Technologies Used

ESP32 Arduino C++ Mecanum Drive IR Sensors
